# About

> [!abstract] Scope and Summary
> This project focuses on curriculum-driven budgeting and donor advocacy in a school library setting. The project includes creating a curriculum map, developing a budget proposal using a zero-based approach, identifying a potential donor, and crafting a pitch statement and mock social media flyer to advocate for funding.
- **AASL Standard:** This project aligns with **ALA/AASL Standard 5.2: Leadership and Collaboration.**
    - This standard highlights my ability to advocate for my library’s needs through budgeting and justifying the program’s needs through the budget.
    - The project requires the me to engage in budgeting, donor advocacy, and creating a pitch for funding.
    - This activity demonstrates networking and collaborating with the larger community in order to achieve the best possible outcome for my students in this media program.
