### Addressing a Diverse School of Learners

---

> [!abstract] Scope and Summary
> This assignment provides an overview of Waterloo Elementary School. It includes details about the school's demographics, academic achievements, staff, and library media program.
- **InTASC Principle**: This project aligns with **InTASC Standard #3: Learning Environments**.
    - This principle focuses on creating environments that support individual and collaborative learning, positive social interaction, and active engagement.
    - By providing an overview of the school's demographics, programs, and support services, the document gives context to the learning environment. This sets up any future project where an instructional leadership team can use to further enhance the learning environment in school since the overview gives relevant statistics and context.
    - It indirectly addresses how the school strives to create a supportive environment for all students through its academic programs and support for diverse learners.
